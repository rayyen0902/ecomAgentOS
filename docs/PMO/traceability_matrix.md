# PMO 需求追踪矩阵

> **文档定位：** 将原设计文档的功能点映射到实施工单与阶段  
> **维护频率：** 每个工单验收后更新  
> **文档版本：** v1.0

---

## 一、矩阵说明

| 字段 | 含义 |
|------|------|
| `原文档章节` | 原设计文档中的章节号或位置 |
| `功能模块` | 业务模块分类 |
| `功能点` | 具体功能描述 |
| `实现阶段` | Phase 0 ~ Phase 7 |
| `工单编号` | 对应 PMO 工单 |
| `当前状态` | 未开始 / 开发中 / 待验收 / 已验收 / 已废弃 |
| `备注` | 与原设计的偏差、风险、优先级说明 |
