## 六、LLM接入架构

### 统一抽象层

```python
# backend/core/llm/provider.py
class LLMProvider(ABC):
    async def chat(self, messages, model=None, **kwargs) -> str: pass
    async def generate_image(self, prompt, size="1024x1024") -> str: pass
    async def generate_video(self, prompt, **kwargs) -> dict: pass
    async def get_video_result(self, video_id: str) -> dict: pass

class AgnesProvider(LLMProvider):
    DEFAULT_TEXT_MODEL = "agnes-2.0-flash"
    DEFAULT_IMAGE_MODEL = "agnes-image-2.1-flash"
    DEFAULT_VIDEO_MODEL = "agnes-video-v2.0"
```

### 切换方式

```bash
# .env
LLM_PROVIDER=agnes          # agnes | openai | deepseek
LLM_API_KEY=sk-xxx
LLM_BASE_URL=https://apihub.agnes-ai.com/v1
```

**切换成本 = 改3行环境变量 + 重启。**

## 九、安全基线

| 能力 | 实现 |
|------|------|
| API认证 | JWT Bearer Token（access 24h + refresh 7d） |
| 密码存储 | bcrypt哈希（salt=12） |
| Cookie加密 | AES-256-GCM（32字节密钥） |
| 买家信息加密 | AES-256-GCM |
| API限流 | slowapi（登录10/分钟，默认100/分钟） |
| 数据库隔离 | PostgreSQL RLS（tenant_id过滤） |
| 日志脱敏 | 买家信息/密码/API密钥不打印 |

## 十、关键风险与应对

| 风险 | 影响 | 应对 |
|------|------|------|
| LangGraph 1.x API变化 | Agent无法运行 | 使用官方文档最新示例，减少定制 |
